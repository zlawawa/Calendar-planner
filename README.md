## Календарь + планировщик задач<br>

### Суть проекта: <br>
Консольное приложение для учёта личных дел и событий, которое объединяет календарь и планировщик задач. Позволяет создавать, хранить и просматривать задачи, привязанные к конкретной дате и времени.<br><br>

### По компонентам: <br>
1. **Календарь** - статичное визуальное представление структуры месяца (дни, недели).
2. **Планировщик задач** - инструмент для создания и организации задач с привязкой к дате и времени.<br><br>

### Ключевой функционал: <br>
1. Расчет дня недели для любой введенной даты
2. Отображение календаря на любой месяц
3. Добавление/удаление/редактирование задач
4. Поиск задач по дате
5. Сортировка списка задач по дате и времени
6. Сохранение данных (списка задач) в файл
7. Загрузка данных (списка задач) из файла
---
## Структура файлов
```
Project_Module.h          - заголовочный файл
Project_Main.cpp          - точка входа
Project_Module.cpp        - реализация логики
FinalListOfEntries.txt    - файл для сохранения списка задач
OriginalListOfEntries.txt - файл для загрузки списка задач в память программы
README.md                 - описание проекта
```
