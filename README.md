## Календарь + планировщик задач<br>
Это единая цифровая система управления временем, которая совмещает фиксированные события с задачами.<br><br>

### По компонентам: <br>
1. **Календарь** - система структурирования времени (дни, недели, месяцы) предназначенная для отслеживания и планирования событий.
2. **Планировщик задач** - инструмент для создания, организации и отслеживания выполнения задач.<br><br>

### Суть проекта: <br>
Консольное приложение для управления личным временем, которое объединяет календарь событий и планировщик задач. Позволяет создавать и просматривать задачи, привязанные к конкретной дате и времени, в едином интерфейсе.<br><br>

### Ключевой функционал: <br>
1. Расчет дня недели для любой введенной даты
2. Отображение календаря на любой месяц
3. Добавление/удаление/редактирование задач
4. Поиск задач по дате
5. Сортировка списка задач по дате и времени
6. Сохранение данных (списка задач) в файл
7. Загрузка данных (списка задач) из файла
